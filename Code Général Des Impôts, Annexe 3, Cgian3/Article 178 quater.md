Article 178 quater
----
Les titres de mouvement accompagnant les vins circulant sur le territoire
français doivent comporter les indications suivantes :

a. L'Etat sur le territoire duquel les raisins ont été récoltés et la
vinification a eu lieu ;

b. La zone viticole dans laquelle les raisins ont été récoltés ;

c. La couleur ; d. La mention "V.Q.P.R.D." ou "Vin de table" ;

e. Le titre alcoométrique volumique ;

f. Lorsque les vins ont fait l'objet d'une opération de coupage, la mention "Vin
issu de coupage" ou "Coupage".
