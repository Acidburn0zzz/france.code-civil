Article 178 quinquies
----
Les marchands en gros et les producteurs qui effectuent des opérations de
coupage doivent tenir un compte d'entrées et sorties distinct [*obligation
comptable*] :

Pour les vins provenant de chaque Etat ;

Pour les vins provenant de chaque zone viticole ;

Pour les V.Q.P.R.D. ;

Pour les vins de table ;

Pour les vins ayant fait l'objet d'opérations de coupage.
